# Manual test — AI Node Editor Stage 4 Authoring

Дата: 2026-07-09  
Ветка: `real-wargame-preview`  
Назначение: проверить, что редактор нод уже не только видимый, но умеет менять рабочий граф.

## Запуск

Запустить двойным кликом:

```text
Run-AI-Node-Editor.bat
```

Откроется:

```text
http://127.0.0.1:5173/ai-node-editor.html
```

## Быстрая проверка

1. Нажать сверху:

```text
Auto check 4–5
```

Ожидание:

```text
Point 4 OK / Пункт 4 OK
Point 5 OK / Пункт 5 OK
```

2. Если автопроверка прошла, перейти к authoring-проверке ниже.

## Authoring-проверка

### 1. Добавить ноду

Слева в палитре нажать любую ноду, например:

```text
Danger Above Threshold
```

Ожидание:

```text
новая нода появилась на графе;
она выбрана;
справа открыт её инспектор.
```

### 2. Перетащить ноду

Зажать новую ноду мышью, перетащить и отпустить.

Ожидание:

```text
нода переместилась;
линии графа обновились после отпускания;
позиция запомнилась в браузере.
```

### 3. Связать ноды

Выбрать родительскую ноду на графе. В инспекторе справа выбрать child node в выпадающем списке и нажать:

```text
Link selected → child
```

Ожидание:

```text
в списке children появилась новая связь;
на графе появилась линия.
```

### 4. Изменить параметры

В инспекторе изменить, например:

```json
{
  "threshold": 45
}
```

Нажать:

```text
Save node
```

Ожидание:

```text
сообщение Saved node;
Graph JSON preview снизу показывает новое значение.
```

### 5. Проверить изменённый граф через local engine

Нажать:

```text
Validate graph
```

Ожидание:

```text
validation.valid = true
```

Если граф сломан, ожидаемая нормальная ошибка — русско/английское сообщение с nodeId.

### 6. Экспортировать JSON

Нажать:

```text
Export JSON
```

Ожидание:

```text
браузер скачал soldier_default_survival_graph.json или soldier_custom_graph.json;
в файле есть английский base и русский overlay: displayName + displayNameRu.
```

### 7. Импортировать JSON обратно

Нажать:

```text
Import JSON
```

Выбрать экспортированный файл.

Ожидание:

```text
граф загрузился обратно;
ноды отображаются;
Validate graph всё ещё проходит.
```

### 8. Сбросить к базовому графу

Нажать:

```text
Reset
```

Ожидание:

```text
граф вернулся к bundled Soldier Survival Graph.
```

## Что этот этап ещё НЕ делает

```text
не подключает граф к живому SimulationTick;
не двигает настоящего солдата на карте;
не сохраняет JSON прямо в repo-файлы;
не делает красивое соединение портами мышью;
не делает WebSocket;
не делает worker_threads.
```

## Что прислать в чат при ошибке

1. Текст из окна `Run-AI-Node-Editor.bat`.
2. Что написано внизу в `Validation / Engine result`.
3. Что именно делал: add / drag / link / save / export / import.
4. Если ошибка после `Validate graph`, прислать JSON ошибки из нижнего окна.
