# Единый игровой редактор сцены — план внедрения

> **Для исполнителей:** выполнять по пунктам в `real-wargame-preview`; `main` не трогать.

**Цель:** дать пользователю единый встроенный редактор, где параметры предмета, бойца, угрозы и рельефа задаются до создания и применяются к выбранным экземплярам.

**Архитектура:** шаблоны создания хранятся отдельно от сцены в `GameEditorDrafts`. `GameEditorPlacement` создаёт полноценные экземпляры из шаблонов. `GameEditorWorkbench` отвечает только за интерфейс и применение шаблона к выбранным объектам. `BoardInputController` направляет клик либо в постановку настроенного экземпляра, либо в старую систему выбора и трансформации.

**Стек:** TypeScript, DOM, PixiJS 7, Vite.

## Ограничения

- Только ветка `real-wargame-preview`.
- Не ломать экспорт/импорт существующих сцен.
- Не смешивать редактор сцены с редактором AI-нод.
- Не удалять существующие функции выбора и перемещения без замены.
- Визуальную проверку не считать выполненной без запуска настоящей игры.
