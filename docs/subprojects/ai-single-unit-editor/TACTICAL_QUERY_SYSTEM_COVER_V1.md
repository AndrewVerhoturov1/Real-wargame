# Tactical Query System — поиск укрытия Graph v2

## Что было до изменения

Нода `FindBestObject` передавала поиск мосту игры, а мост вызывал `findBestCoverForThreat()`. Эта функция в одном цикле одновременно создавала варианты, отбрасывала их, считала одну скрытую формулу и сразу возвращала победителя. Graph v2 не видел кандидатов и не мог объяснить выбор.

## Что сделано

Новый запрос принадлежит одному запуску графа и проходит четыре явные ноды:

1. **Создать кандидаты укрытий** — получает низкоуровневые факты о позициях и применяет бюджет.
2. **Фильтр тактических позиций** — фиксирует причины исключения.
3. **Оценить позиции** — считает отдельные компоненты мягкой оценки.
4. **Выбрать лучшую позицию** — единственное место, которое может записать `best_cover_position`.

Пустой граф не вызывает генератор. Граф без четвёртой ноды сохраняет диагностику запроса, но не создаёт `best_cover_position`.

## Данные кандидата

Для каждого кандидата сохраняются позиция и источник; нахождение на карте; наличие точного маршрута; дистанция; защита от текущего направления угрозы; защита и маскировка; опасность маршрута; прямой, обратный или нейтральный склон; соответствие текущему приказу; итоговая оценка и шесть отдельных компонентов; все причины исключения.

## Бюджет

Нода генерации редактирует максимальное число кандидатов, радиус поиска и максимальное время расчёта. При досрочной остановке запрос хранит машинный код, английское объяснение и русское объяснение.

## Граница ответственности

`CoverEvaluation.findBestCoverForThreat()` сохранена для совместимости как старый низкоуровневый помощник, но живой мост Graph v2 больше её не вызывает. Мост игры только сообщает факты. Фильтрация, веса и выбор принадлежат нодам графа.

## Диагностика

Панель «След ИИ» показывает состояние и бюджет запроса, всех кандидатов, победителя, исключённые позиции и причины, отдельные составляющие оценки и причину досрочной остановки.

## Автоматическая проверка

Финальный проверенный коммит должен пройти:

- `npm run tactical-query:smoke`
- `npm run tactical-query-ui:smoke`
- `npm run graph-v2:smoke`
- `npm run runtime:smoke`
- `npm run pathfinding:smoke`
- `npm run directional-terrain:smoke`
- `npm run validate:ai-graph`
- `npm run docs:sync`
- `npm run build`
