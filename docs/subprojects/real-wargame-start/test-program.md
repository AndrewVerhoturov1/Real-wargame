# Программа проверки — рельеф высот v2.1

Цель проверки: заменить некрасивые подписи `+1`, `+2`, `-1` на обычной карте мягким визуальным слоем рельефа. Данные высоты остаются клеточными для будущих расчётов солдат, но игрок должен видеть не таблицу цифр, а понятные пятна низин, холмов и гребней.

## Что открыть

Запускать из папки предпросмотра:

```text
scripts/windows/run-preview.bat
```

Или двойным кликом по корневому файлу:

```text
Run-Real-Wargame.bat
```

## Что проверить руками

1. Проект открывается в браузере.
2. Текст интерфейса по умолчанию русский.
3. Сетка видна по умолчанию.
4. Кнопка сверху должна показывать `Обзор: выкл`.
5. Новая кнопка сверху должна показывать `Цифры высоты: выкл`.
6. В обычном виде на карте не должно быть постоянных надписей `+1`, `+2`, `-1` по клеткам.
7. Рельеф должен читаться цветом и мягкими пятнами:
   - низины темнее/холоднее;
   - обычная земля нейтральная;
   - холмы и гребни светлее/суше;
   - форма не должна выглядеть как ровная таблица квадратов.
8. Возле наблюдательного поста должна быть заметная более высокая местность.
9. В болотистой/водной зоне должны быть заметны более низкие участки.
10. Нажать `Цифры высоты: выкл`.
11. Кнопка должна стать `Цифры высоты: вкл`.
12. После включения должны появиться отладочные цифры высоты.
13. Нажать кнопку ещё раз.
14. Цифры высоты должны исчезнуть, а мягкий слой рельефа остаться.
15. Круги зон, линии, предметы и рамки должны оставаться мягкими: сглаживание не должно быть выключено.
16. Покрутить колесо мыши для зума 10–20 секунд.
17. Зум не должен возвращаться к длинной плавной анимации через промежуточные кадры.
18. Текст не должен снова начать заметно дёргаться.
19. Земля, рельеф и сетка не должны мигать при обычном зуме.
20. Внизу слева нажать плавающую кнопку `Редактор`.
21. После включения редактора игровые разделы должны скрыться.
22. Справа должен остаться отдельный раздел `Инспектор редактора`.
23. В режиме редактора подписи предметов и юнитов на карте должны быть скрыты.
24. Выбрать инструмент `Выбрать / тянуть`.
25. Кликнуть по предмету на карте: предмет должен подсветиться рамкой.
26. Зажать левую кнопку мыши в центре/на теле предмета и потянуть: предмет должен перемещаться, а не растягиваться.
27. Выбрать инструмент `Создать зону`.
28. Кликнуть по карте: новая зона должна появиться в точке клика и стать выбранной.
29. Выбрать инструмент `Выбрать / тянуть`.
30. Кликнуть по зоне: зона должна подсветиться рамкой/ручками.
31. Зажать левую кнопку мыши внутри зоны и потянуть: зона должна перемещаться.
32. Нажать `Скачать JSON сцены`.
33. Браузер должен скачать файл с именем примерно `real-wargame-scene-...json`.
34. В скачанном JSON должны быть поля:
    - `map.heightMap`;
    - `map.forestMap`.
35. Нажать `Скачать отчёт производительности`.
36. Браузер должен скачать файл с именем примерно `real-wargame-performance-...json`.
37. В отчёте производительности поле `renderer.antialias` должно быть `true`.
38. В отчёте производительности поле `renderer.maxFPS` должно быть `60`.
39. В отчёте производительности поле `renderer.mapRender` должно говорить про `cached canvas elevation texture`.
40. В отчёте производительности поле `renderer.heightLabels` должно быть `false`, если ты не включал кнопку `Цифры высоты`.

## Ожидаемый результат

```text
Карта:
- высоты поддерживают уровни -2, -1, 0, 1, 2, 3, 4;
- обычный вид карты больше не засыпан цифрами +1/+2/-1;
- цифры высоты включаются только отдельной отладочной кнопкой;
- высоты видны как мягкий слой рельефа поверх базовой земли;
- данные остаются клеточными, чтобы потом использовать их для обзора, укрытий и движения солдат.

Рендер:
- базовая земля остаётся пачками по типам местности;
- рельеф собирается в отдельный canvas/texture-слой;
- сетка, предметы, зоны, юниты и приказы остаются отдельными слоями;
- сглаживание PixiJS остаётся включённым.

Сохранение:
- экспорт сцены теперь версия scene-export-v2;
- JSON содержит heightMap;
- JSON содержит forestMap как заготовку под будущие слои леса: 0 нет леса, 1 редкий лес, 2 густой лес.
```

## GO / NO-GO

```text
GO, если:
- рельеф читается лучше, чем цифры на клетках;
- нет ощущения квадратной таблицы;
- цифры высоты выключены по умолчанию и включаются только кнопкой;
- редактор предметов и зон работает;
- экспорт JSON сцены работает;
- зум не стал заметно хуже.

NO-GO, если:
- карта стала выглядеть грязно или непонятно;
- рельеф всё ещё читается как квадраты;
- пропали юниты, зоны, предметы или линии приказов;
- кнопка цифр высоты не работает;
- экспорт сцены сломался;
- зум стал заметно хуже.
```

## Что не входит в этот пакет

```text
- кисти высоты в редакторе;
- кисти редкого/густого леса;
- автоматическая запись JSON прямо в файлы проекта;
- загрузка скачанного JSON обратно через интерфейс;
- влияние высоты на обзор, стрельбу и укрытия;
- влияние леса на видимость, скорость и скрытность;
- полноценный поиск пути.
```
