# Программа проверки — экспорт сцены и плавный зум v1.2

Цель проверки: убедиться, что текущую расстановку из редактора можно скачать в JSON-файл, а зум стал менее резким. Экспорт пока не переписывает файлы проекта сам: браузер скачивает JSON, который потом можно отдать Codex для закрепления в исходных файлах.

## Что открыть

Запускать из папки предпросмотра:

```text
scripts/windows/run-preview.bat
```

Или двойным кликом по корневому файлу:

```text
Run-Real-Wargame.bat
```

## Что проверить руками

1. Проект открывается в браузере.
2. Текст интерфейса по умолчанию русский.
3. Сетка видна по умолчанию.
4. В левом верхнем углу карты видна подпись масштаба: `1 клетка = 10 м`.
5. Карта визуально компактная.
6. Юниты заметно меньше клетки и не выглядят как круг шириной 10 метров.
7. Покрутить колесо мыши для зума: масштаб должен меняться плавнее, не одним резким скачком.
8. Внизу слева нажать плавающую кнопку `Редактор`.
9. После включения редактора игровые разделы должны скрыться.
10. Справа должен остаться отдельный раздел `Инспектор редактора`.
11. В режиме редактора подписи предметов и юнитов на карте должны быть скрыты, чтобы редактор меньше тормозил.
12. В инспекторе редактора должен быть блок `Сохранение`.
13. В блоке `Сохранение` должна быть кнопка `Скачать JSON сцены`.
14. Выбрать инструмент `Выбрать / тянуть`.
15. Кликнуть по предмету на карте: предмет должен подсветиться рамкой.
16. Зажать левую кнопку мыши в центре/на теле предмета и потянуть: предмет должен перемещаться, а не растягиваться.
17. Во время перетаскивания предмета земля и сетка не должны мигать или пересоздаваться визуально.
18. Зажать левую кнопку мыши именно на квадратной ручке предмета и потянуть: размер предмета должен меняться.
19. Зажать левую кнопку мыши на круглой ручке над рамкой и потянуть: предмет должен вращаться.
20. Выбрать инструмент `Создать зону`.
21. Кликнуть по карте: новая зона должна появиться в точке клика и стать выбранной.
22. Выбрать инструмент `Выбрать / тянуть`.
23. Кликнуть по зоне: зона должна подсветиться рамкой/ручками.
24. Зажать левую кнопку мыши внутри зоны и потянуть: зона должна перемещаться.
25. Потянуть за ручку на границе зоны: размер зоны должен измениться.
26. Нажать `Скачать JSON сцены`.
27. Браузер должен скачать файл с именем примерно `real-wargame-scene-...json`.
28. Внутри файла должны быть блоки:
    - `map`;
    - `units`;
    - `pressureZones`.
29. В `map.objects` должны быть текущие положения, размеры и повороты предметов.
30. В `pressureZones` должны быть текущие положения, размеры, опасность и стресс зон.
31. В `units` должны быть текущие положения юнитов.
32. Кнопка `Очистить всё` должна после подтверждения удалить все предметы, всех юнитов и все зоны на карте.
33. Выключить редактор кнопкой `Редактор: вкл/выкл` или плавающей кнопкой.
34. После выключения редактора справа снова должны появиться игровые разделы.
35. После выключения редактора обычное управление должно вернуться:
    - левый клик выбирает юнита;
    - правый клик отдаёт приказ движения.

## Ожидаемый результат

```text
Сохранение:
- редактор может скачать текущую сцену в JSON;
- JSON содержит карту, предметы, юниты и зоны;
- это ещё не автоматическая запись в файлы проекта;
- скачанный файл можно отдать Codex для закрепления изменений.

Зум:
- колесо мыши задаёт целевой масштаб;
- камера плавно подтягивается к нему;
- зум должен быть менее резким.

Плавность:
- земля и сетка не пересобираются при каждом движении предмета;
- при перетаскивании предмета обновляется только слой предметов;
- в режиме редактора подписи предметов и юнитов скрыты;
- лишние HTML-подписи не переписываются каждый кадр.
```

## Что не входит в этот пакет

```text
- автоматическая запись JSON прямо в файлы проекта;
- загрузка скачанного JSON обратно через интерфейс;
- перенос симуляции в отдельный рабочий поток;
- физическая коллизия с деревьями, домами, зонами и предметами;
- полноценный поиск пути.
```
