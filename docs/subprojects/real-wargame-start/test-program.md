# Программа проверки — параметры поведения v0.3

Цель проверки: убедиться, что ветка предпросмотра показывает паспорт солдата и считает параметры для будущего поведения, но пока не заставляет юнита застревать, ложиться или останавливаться.

## Что открыть

Запускать из папки предпросмотра:

```text
scripts/windows/run-preview.bat
```

Или двойным кликом по корневому файлу:

```text
Run-Real-Wargame.bat
```

## Что проверить руками

1. Проект открывается в браузере.
2. Текст интерфейса по умолчанию русский.
3. Сетка визуально скрыта.
4. Сектор обзора визуально скрыт.
5. На карте видны полупрозрачные рыжие зоны параметров.
6. Левым кликом выбирается юнит.
7. В правой панели появляется `Инспектор поведения`.
8. Под инспектором появляется блок `Паспорт солдата`.
9. В паспорте видны постоянные качества:
   - устойчивость;
   - осторожность;
   - решительность;
   - дисциплина;
   - инициативность;
   - тактика;
   - владение оружием.
10. В паспорте видны текущие состояния:
   - усталость;
   - стресс;
   - боевой дух;
   - растерянность;
   - здоровье.
11. В паспорте видны восприятие и движение:
   - внимательность;
   - обзор;
   - интуиция;
   - скорость;
   - скрытность.
12. Под инспектором видны кнопки положения:
   - `▮ Стоя`;
   - `▰ Пригнулся`;
   - `━ Лежит`.
13. Выбрать юнита и нажать каждую кнопку положения.
14. Маркер юнита на карте должен менять вид:
   - стоя — обычный круг;
   - пригнулся — меньший круг;
   - лежит — низкий вытянутый маркер.
15. Правым кликом дать юниту приказ пройти через зону параметров.
16. Пока юнит реально входит в зону:
   - опасность становится больше 0;
   - стресс растёт;
   - юнит продолжает идти и не застревает;
   - положение не должно насильно менять движение;
   - состояние не должно останавливать движение.
17. Когда юнит выходит из зоны:
   - опасность падает до 0;
   - стресс постепенно снижается.
18. Выбрать разных юнитов и убедиться, что значения паспорта отличаются по профилю.
19. Групповое выделение рамкой продолжает работать.

## Ожидаемый результат

```text
Это ещё не поведение.
Это слой параметров солдата:
- у юнита есть постоянные качества;
- у юнита есть текущие состояния;
- у юнита есть параметры восприятия и движения;
- зона считает опасность и стресс;
- паспорт солдата показывает числа;
- кнопки вручную меняют положение юнита;
- юнит не застревает в зоне;
- движение не блокируется стрессом;
- настоящее поведение будем задавать позже.
```

## Что не входит в этот пакет

```text
- автоматическое приседание;
- автоматическое положение лёжа;
- остановка от стресса;
- поиск укрытий;
- таблица оценки вариантов действий;
- память и знания юнита;
- полноценный план командира.
```
