# Программа проверки — кэш слоёв и зум v1.3

Цель проверки: убедиться, что зум и редактор стали легче. Земля/сетка теперь не просто отделены от предметов, но и кэшируются как неподвижный слой. Зоны параметров тоже отделены от динамической подсветки мыши и не пересобираются без изменения.

## Что открыть

Запускать из папки предпросмотра:

```text
scripts/windows/run-preview.bat
```

Или двойным кликом по корневому файлу:

```text
Run-Real-Wargame.bat
```

## Что проверить руками

1. Проект открывается в браузере.
2. Текст интерфейса по умолчанию русский.
3. Сетка видна по умолчанию.
4. В левом верхнем углу карты видна подпись масштаба: `1 клетка = 10 м`.
5. Карта визуально компактная.
6. Юниты заметно меньше клетки и не выглядят как круг шириной 10 метров.
7. Покрутить колесо мыши для зума: масштаб должен меняться плавнее, не одним резким скачком.
8. Во время зума земля и сетка не должны мигать или пересоздаваться визуально.
9. Во время зума рыжие зоны не должны мигать или пересоздаваться визуально, если сами зоны не редактируются.
10. Внизу слева нажать плавающую кнопку `Редактор`.
11. После включения редактора игровые разделы должны скрыться.
12. Справа должен остаться отдельный раздел `Инспектор редактора`.
13. В режиме редактора подписи предметов и юнитов на карте должны быть скрыты, чтобы редактор меньше тормозил.
14. В инспекторе редактора должен быть блок `Сохранение`.
15. В блоке `Сохранение` должна быть кнопка `Скачать JSON сцены`.
16. Выбрать инструмент `Выбрать / тянуть`.
17. Кликнуть по предмету на карте: предмет должен подсветиться рамкой.
18. Зажать левую кнопку мыши в центре/на теле предмета и потянуть: предмет должен перемещаться, а не растягиваться.
19. Во время перетаскивания предмета земля и сетка не должны мигать или пересоздаваться визуально.
20. Зажать левую кнопку мыши именно на квадратной ручке предмета и потянуть: размер предмета должен меняться.
21. Зажать левую кнопку мыши на круглой ручке над рамкой и потянуть: предмет должен вращаться.
22. Выбрать инструмент `Создать зону`.
23. Кликнуть по карте: новая зона должна появиться в точке клика и стать выбранной.
24. Выбрать инструмент `Выбрать / тянуть`.
25. Кликнуть по зоне: зона должна подсветиться рамкой/ручками.
26. Зажать левую кнопку мыши внутри зоны и потянуть: зона должна перемещаться.
27. Потянуть за ручку на границе зоны: размер зоны должен измениться.
28. Нажать `Скачать JSON сцены`.
29. Браузер должен скачать файл с именем примерно `real-wargame-scene-...json`.
30. Внутри файла должны быть блоки:
    - `map`;
    - `units`;
    - `pressureZones`.
31. В `map.objects` должны быть текущие положения, размеры и повороты предметов.
32. В `pressureZones` должны быть текущие положения, размеры, опасность и стресс зон.
33. В `units` должны быть текущие положения юнитов.
34. Кнопка `Очистить всё` должна после подтверждения удалить все предметы, всех юнитов и все зоны на карте.
35. Выключить редактор кнопкой `Редактор: вкл/выкл` или плавающей кнопкой.
36. После выключения редактора справа снова должны появиться игровые разделы.
37. После выключения редактора обычное управление должно вернуться:
    - левый клик выбирает юнита;
    - правый клик отдаёт приказ движения.

## Ожидаемый результат

```text
Кэш слоёв:
- земля, сетка, подпись масштаба и граница карты кэшируются как неподвижный слой;
- при зуме браузер масштабирует готовый слой, а не тысячи отдельных клеток;
- зоны параметров вынесены в отдельный слой;
- подсветка клетки мыши и рамка выделения не заставляют пересобирать зоны.

Зум:
- колесо мыши задаёт целевой масштаб;
- камера плавно подтягивается к нему;
- зум должен быть менее резким и менее тяжёлым.

Сохранение:
- редактор может скачать текущую сцену в JSON;
- JSON содержит карту, предметы, юниты и зоны;
- это ещё не автоматическая запись в файлы проекта;
- скачанный файл можно отдать Codex для закрепления изменений.
```

## Что не входит в этот пакет

```text
- автоматическая запись JSON прямо в файлы проекта;
- загрузка скачанного JSON обратно через интерфейс;
- перенос симуляции в отдельный рабочий поток;
- физическая коллизия с деревьями, домами, зонами и предметами;
- полноценный поиск пути.
```
