# Программа проверки — редактор карты v0.5

Цель проверки: убедиться, что ветка предпросмотра показывает карту в понятном масштабе, рисует объекты видом сверху, разводит юнитов при совпадении точки назначения и даёт простой режим редактора для предметов и юнитов.

## Что открыть

Запускать из папки предпросмотра:

```text
scripts/windows/run-preview.bat
```

Или двойным кликом по корневому файлу:

```text
Run-Real-Wargame.bat
```

## Что проверить руками

1. Проект открывается в браузере.
2. Текст интерфейса по умолчанию русский.
3. Сетка видна по умолчанию.
4. В левом верхнем углу карты видна подпись масштаба: `1 клетка = 10 м`.
5. По сетке понятно, что один квадрат карты равен 10 метрам.
6. Сектор обзора визуально скрыт.
7. На карте видны полупрозрачные рыжие зоны параметров.
8. Предметы на карте выглядят как вид сверху, а не сбоку.
9. Размеры объектов выглядят соразмерно сетке.
10. Правая панель не уходит безвозвратно ниже экрана: её можно прокручивать колесом мыши.
11. В правой панели есть сворачиваемые разделы:
    - `Инспектор поведения`;
    - `Положение юнита`;
    - `Паспорт солдата`;
    - `Редактор карты`.
12. Открыть раздел `Редактор карты`.
13. Нажать `Редактор: вкл`.
14. Выбрать инструмент `Выбрать`, кликнуть по предмету на карте: предмет должен подсветиться рамкой, а в редакторе должен появиться его id.
15. Изменить ширину, высоту или поворот предмета и нажать `Применить к выбранному предмету`: выбранный предмет должен измениться на карте.
16. Выбрать инструмент `Создать предмет`, выбрать тип предмета и кликнуть по карте: новый предмет должен появиться в точке клика.
17. Выбрать инструмент `Создать юнит`, выбрать тип юнита и кликнуть по карте: новый юнит должен появиться в точке клика и стать выбранным.
18. Выбрать инструмент `Удалить`, кликнуть по предмету или юниту: объект должен исчезнуть.
19. Кнопка `Удалить выбранное` должна удалять выбранный предмет или выбранный юнит.
20. Выключить редактор кнопкой `Редактор: вкл/выкл`.
21. После выключения редактора обычное управление должно вернуться:
    - левый клик выбирает юнита;
    - правый клик отдаёт приказ движения.
22. Отправить одного юнита в точку правым кликом: он должен дойти до точки.
23. Отправить двух юнитов в одну и ту же точку: они не должны полностью слипнуться в один маркер, а должны сохранить небольшое расстояние друг от друга.
24. Проверка коллизии: юниты могут подходить близко, потому что считается только центральная треть тела, но центр одного юнита не должен полностью накладываться на центр другого.
25. Правым кликом дать юниту приказ пройти через зону параметров.
26. Пока юнит реально входит в зону:
    - опасность становится больше 0;
    - стресс растёт;
    - юнит продолжает идти и не застревает;
    - положение не должно насильно менять движение;
    - состояние не должно останавливать движение.
27. Когда юнит выходит из зоны:
    - опасность падает до 0;
    - стресс постепенно снижается.
28. Выбрать разных юнитов и убедиться, что значения паспорта отличаются по профилю.
29. Групповое выделение рамкой продолжает работать, когда редактор выключен.

## Ожидаемый результат

```text
Это ещё не полноценный редактор миссий.
Это простой отладочный редактор карты:
- карта имеет масштаб 1 клетка = 10 м;
- сетка видна и обозначает метры;
- объекты отрисованы как вид сверху;
- предметы можно выбирать;
- выбранный предмет подсвечивается;
- выбранному предмету можно менять ширину, высоту и поворот;
- можно создавать предметы кликом по карте;
- можно создавать юнитов кликом по карте;
- можно удалять предметы и юнитов;
- при выключенном редакторе обычное управление возвращается;
- юниты не слипаются при одной точке назначения;
- коллизия считается только маленьким центральным ядром юнита;
- правая панель прокручивается;
- паспорт солдата показывает числа;
- зоны считают опасность и стресс.
```

## Что не входит в этот пакет

```text
- сохранение изменений редактора в JSON-файл;
- загрузка пользовательской карты из редактора;
- перетаскивание объектов мышью;
- физическая коллизия с деревьями, домами и предметами;
- автоматическое приседание;
- автоматическое положение лёжа;
- остановка от стресса;
- поиск укрытий;
- полноценный поиск пути;
- таблица оценки вариантов действий;
- память и знания юнита;
- полноценный план командира.
```
