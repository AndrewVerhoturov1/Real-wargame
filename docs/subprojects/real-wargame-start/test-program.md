# Программа проверки — стабильный текст при зуме v1.4

Цель проверки: убрать дрожание текста при зуме. Подпись масштаба вынесена из масштабируемой карты в отдельную неподвижную HTML-плашку, HTML-подписи карты ставятся только в целые пиксели, правый интерфейс больше не использует размытый фон поверх движущейся карты, а строка масштаба в инспекторе обновляется реже.

## Что открыть

Запускать из папки предпросмотра:

```text
scripts/windows/run-preview.bat
```

Или двойным кликом по корневому файлу:

```text
Run-Real-Wargame.bat
```

## Что проверить руками

1. Проект открывается в браузере.
2. Текст интерфейса по умолчанию русский.
3. Сетка видна по умолчанию.
4. В левом верхнем углу карты видна неподвижная подпись масштаба: `1 клетка = 10 м`.
5. Покрутить колесо мыши для зума: масштаб должен меняться плавнее, не одним резким скачком.
6. Во время зума подпись масштаба не должна увеличиваться/уменьшаться вместе с картой.
7. Во время зума подпись масштаба не должна дрожать.
8. Во время зума правый интерфейс не должен заметно мерцать или давать эффект “тумана”.
9. Во время зума текстовые подписи на карте должны меньше прыгать, потому что их координаты округляются до целых пикселей.
10. Во время зума земля и сетка не должны мигать или пересоздаваться визуально.
11. Во время зума рыжие зоны не должны мигать или пересоздаваться визуально, если сами зоны не редактируются.
12. Внизу слева нажать плавающую кнопку `Редактор`.
13. После включения редактора игровые разделы должны скрыться.
14. Справа должен остаться отдельный раздел `Инспектор редактора`.
15. В режиме редактора подписи предметов и юнитов на карте должны быть скрыты, чтобы редактор меньше тормозил.
16. В инспекторе редактора должен быть блок `Сохранение`.
17. В блоке `Сохранение` должна быть кнопка `Скачать JSON сцены`.
18. Выбрать инструмент `Выбрать / тянуть`.
19. Кликнуть по предмету на карте: предмет должен подсветиться рамкой.
20. Зажать левую кнопку мыши в центре/на теле предмета и потянуть: предмет должен перемещаться, а не растягиваться.
21. Во время перетаскивания предмета земля и сетка не должны мигать или пересоздаваться визуально.
22. Выбрать инструмент `Создать зону`.
23. Кликнуть по карте: новая зона должна появиться в точке клика и стать выбранной.
24. Выбрать инструмент `Выбрать / тянуть`.
25. Кликнуть по зоне: зона должна подсветиться рамкой/ручками.
26. Зажать левую кнопку мыши внутри зоны и потянуть: зона должна перемещаться.
27. Нажать `Скачать JSON сцены`.
28. Браузер должен скачать файл с именем примерно `real-wargame-scene-...json`.

## Ожидаемый результат

```text
Текст:
- подпись масштаба не масштабируется вместе с картой;
- подпись масштаба не дрожит при зуме;
- правый интерфейс без размытого фона, поэтому текст должен быть стабильнее;
- HTML-подписи карты ставятся в целые пиксели;
- строка масштаба в инспекторе обновляется реже и грубее: 1 знак после запятой.

Карта:
- bitmap-кэш статичной карты отключён, чтобы не было эффекта мутной картинки через туман;
- земля и сетка всё ещё не пересобираются при каждом движении предмета;
- зоны параметров вынесены в отдельный слой;
- подсветка клетки мыши и рамка выделения не заставляют пересобирать зоны.

Сохранение:
- редактор может скачать текущую сцену в JSON;
- JSON содержит карту, предметы, юниты и зоны;
- это ещё не автоматическая запись в файлы проекта.
```

## Что не входит в этот пакет

```text
- автоматическая запись JSON прямо в файлы проекта;
- загрузка скачанного JSON обратно через интерфейс;
- перенос симуляции в отдельный рабочий поток;
- физическая коллизия с деревьями, домами, зонами и предметами;
- полноценный поиск пути.
```
