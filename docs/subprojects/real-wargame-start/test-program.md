# Программа проверки — оптимизация редактора v0.7

Цель проверки: убедиться, что ветка предпросмотра показывает карту в понятном масштабе, даёт интерактивный редактор и работает легче для браузера: карта не пересоздаётся каждый кадр, инспектор и паспорт обновляются реже, но поведение для пользователя остаётся тем же.

## Что открыть

Запускать из папки предпросмотра:

```text
scripts/windows/run-preview.bat
```

Или двойным кликом по корневому файлу:

```text
Run-Real-Wargame.bat
```

## Что проверить руками

1. Проект открывается в браузере.
2. Текст интерфейса по умолчанию русский.
3. Сетка видна по умолчанию.
4. В левом верхнем углу карты видна подпись масштаба: `1 клетка = 10 м`.
5. По сетке понятно, что один квадрат карты равен 10 метрам.
6. Сектор обзора визуально скрыт.
7. На карте видны полупрозрачные рыжие зоны параметров.
8. Предметы на карте выглядят как вид сверху, а не сбоку.
9. Размеры объектов выглядят соразмерно сетке.
10. Без действий пользователя карта не должна заметно подтормаживать только из-за простоя.
11. При движении юнита карта не должна полностью мигать или визуально пересоздаваться.
12. Внизу слева есть отдельная плавающая кнопка `Редактор`.
13. Нажать плавающую кнопку `Редактор`: должен включиться режим редактора и раскрыться раздел `Редактор карты`.
14. В режиме редактора юнитам нельзя отдавать приказы движения правым кликом.
15. В разделе `Редактор карты` есть блок `Слои`:
    - предметы;
    - юниты;
    - зоны параметров.
16. Выключить слой `Предметы`: предметы должны скрыться, выбранный предмет должен сброситься.
17. Включить слой `Предметы`: предметы должны снова появиться.
18. Выключить слой `Юниты`: юниты должны скрыться, выбранный юнит должен сброситься.
19. Включить слой `Юниты`: юниты должны снова появиться.
20. Выключить слой `Зоны параметров`: рыжие зоны должны скрыться.
21. Включить слой `Зоны параметров`: рыжие зоны должны снова появиться.
22. Выбрать инструмент `Выбрать`, кликнуть по предмету на карте: предмет должен подсветиться рамкой.
23. У выбранного предмета должны быть управляющие элементы на карте:
    - рамка;
    - квадратные ручки изменения размера;
    - круглая ручка поворота над рамкой.
24. Кликнуть по квадратной ручке рамки: размер предмета должен увеличиться.
25. Кликнуть по круглой ручке сверху: предмет должен повернуться на 15 градусов.
26. В блоке `Управление выбранным предметом` нажать стрелки: выбранный предмет должен сдвигаться.
27. Нажать `Ширина +`, `Ширина −`, `Высота +`, `Высота −`: размер выбранного предмета должен меняться.
28. Нажать `⟲ 15°` или `⟳ 15°`: выбранный предмет должен поворачиваться.
29. Выбрать инструмент `Переместить`, затем кликнуть по карте: выбранный предмет или выбранный юнит должен переместиться в точку клика.
30. Изменить ширину, высоту или поворот в числах и нажать `Применить числа к выбранному предмету`: выбранный предмет должен измениться на карте.
31. Выбрать инструмент `Создать предмет`, выбрать тип предмета и кликнуть по карте: новый предмет должен появиться в точке клика.
32. Выбрать инструмент `Создать юнит`, выбрать тип юнита и кликнуть по карте: новый юнит должен появиться в точке клика и стать выбранным.
33. Выбрать инструмент `Удалить`, кликнуть по предмету или юниту: объект должен исчезнуть.
34. Кнопка `Удалить выбранное` должна удалять выбранный предмет или выбранный юнит.
35. Кнопка `Очистить всё` должна после подтверждения удалить все предметы и всех юнитов на карте.
36. Выключить редактор кнопкой `Редактор: вкл/выкл` или плавающей кнопкой.
37. После выключения редактора обычное управление должно вернуться:
    - левый клик выбирает юнита;
    - правый клик отдаёт приказ движения.
38. Групповое выделение рамкой продолжает работать, когда редактор выключен.

## Ожидаемый результат

```text
Это ещё не полноценный редактор миссий.
Это оптимизированный браузерный отладочный редактор карты:
- карта не перерисовывается полностью каждый кадр;
- слой карты обновляется только при изменении сетки, предметов, слоя предметов или выбранного предмета;
- правый инспектор обновляется реже, а не 60 раз в секунду;
- паспорт солдата обновляется реже и только при изменении текста;
- у редактора есть отдельная плавающая кнопка режима;
- в режиме редактора юнитам не отдаются приказы движения;
- есть слои предметов, юнитов и зон параметров;
- слои можно скрывать и показывать;
- предметы можно выбирать;
- выбранный предмет подсвечивается рамкой;
- у выбранного предмета есть управляющие ручки на карте;
- размер и поворот можно менять ручками и кнопками;
- выбранный предмет можно сдвигать кнопками;
- выбранный предмет или юнит можно переместить кликом;
- можно создавать предметы кликом по карте;
- можно создавать юнитов кликом по карте;
- можно удалять предметы и юнитов;
- можно очистить все предметы и всех юнитов;
- при выключенном редакторе обычное управление возвращается.
```

## Что не входит в этот пакет

```text
- перенос симуляции в отдельный рабочий поток;
- сохранение изменений редактора в JSON-файл;
- загрузка пользовательской карты из редактора;
- свободное перетаскивание объектов зажатой мышью;
- уменьшение размера через ручки на карте, сейчас ручки увеличивают размер;
- физическая коллизия с деревьями, домами и предметами;
- автоматическое приседание;
- автоматическое положение лёжа;
- остановка от стресса;
- поиск укрытий;
- полноценный поиск пути;
- таблица оценки вариантов действий;
- память и знания юнита;
- полноценный план командира.
```
