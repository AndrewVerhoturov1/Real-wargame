# Test program — Behavior Foundation v0.1

Цель проверки: убедиться, что preview-ветка показывает первый фундамент поведения юнитов перед полноценным ИИ.

## Что открыть

Запускать из preview-папки:

```text
scripts/windows/run-preview.bat
```

Или двойным кликом по корневому файлу:

```text
Run-Real-Wargame.bat
```

## Что проверить руками

1. Проект открывается в браузере.
2. На карте видны полупрозрачные рыжие pressure-зоны.
3. Левым кликом выбирается юнит.
4. Правым кликом можно дать приказ движения.
5. Если правый клик пришёлся внутрь pressure-зоны, выбранный юнит получает осторожный режим:
   - маркер юнита становится ниже/меньше;
   - вокруг юнита появляется рыжий stress-ореол;
   - движение остаётся прямолинейным.
6. Если правый клик пришёлся вне pressure-зоны, юнит возвращается к обычному положению.
7. Групповое выделение рамкой продолжает работать.
8. Переключатели сетки и сектора обзора продолжают работать.

## Ожидаемый результат

```text
Это ещё не полноценный ИИ.
Это первый слой среды для ИИ:
- у юнитов есть behavior profile;
- у юнитов есть state/posture/stress/danger поля;
- карта содержит pressure zones;
- выбранная цель внутри pressure-zone меняет видимое поведение юнита.
```

## Что не входит в этот пакет

```text
- полноценная динамическая реакция во время движения;
- поиск укрытий;
- сложный редактор поведения;
- отдельная панель инспектора решений.
```

Эти пункты должны идти следующими отдельными r-init задачами.
