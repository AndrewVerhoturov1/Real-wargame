# Программа проверки — параметры поведения v0.2

Цель проверки: убедиться, что ветка предпросмотра считает параметры для будущего поведения, но пока не заставляет юнита застревать, ложиться или останавливаться.

## Что открыть

Запускать из папки предпросмотра:

```text
scripts/windows/run-preview.bat
```

Или двойным кликом по корневому файлу:

```text
Run-Real-Wargame.bat
```

## Что проверить руками

1. Проект открывается в браузере.
2. Текст интерфейса по умолчанию русский.
3. Сетка визуально скрыта.
4. Сектор обзора визуально скрыт.
5. На карте видны полупрозрачные рыжие зоны параметров.
6. Левым кликом выбирается юнит.
7. В правой панели появляется `Инспектор поведения`.
8. В инспекторе видны русские подписи:
   - профиль;
   - состояние;
   - положение;
   - действие;
   - опасность;
   - напряжение;
   - причина;
   - причина состояния;
   - причина положения;
   - последнее событие;
   - пороги.
9. Под инспектором видны кнопки положения:
   - `▮ Стоя`;
   - `▰ Пригнулся`;
   - `━ Лежит`.
10. Выбрать юнита и нажать каждую кнопку положения.
11. Маркер юнита на карте должен менять вид:
   - стоя — обычный круг;
   - пригнулся — меньший круг;
   - лежит — низкий вытянутый маркер.
12. Правым кликом дать юниту приказ пройти через зону параметров.
13. Пока юнит реально входит в зону:
   - опасность становится больше 0;
   - напряжение растёт;
   - юнит продолжает идти и не застревает;
   - положение не должно насильно менять движение;
   - состояние не должно останавливать движение.
14. Когда юнит выходит из зоны:
   - опасность падает до 0;
   - напряжение постепенно снижается.
15. Групповое выделение рамкой продолжает работать.

## Ожидаемый результат

```text
Это ещё не поведение.
Это только слой параметров:
- зона считает опасность и напряжение;
- инспектор показывает параметры;
- кнопки вручную меняют положение юнита;
- юнит не застревает в зоне;
- движение не блокируется напряжением;
- настоящее поведение будем задавать позже.
```

## Что не входит в этот пакет

```text
- автоматическое приседание;
- автоматическое положение лёжа;
- остановка от напряжения;
- поиск укрытий;
- таблица оценки вариантов действий;
- память и знания юнита;
- полноценный план командира.
```
