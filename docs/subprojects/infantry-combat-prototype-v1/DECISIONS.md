# Решения

## 2026-07-22 — Физика и поведение разделены

- Код: физика, факты, безопасное исполнение действий.
- Graph v2: выбор действий, пороги, веса и переходы.
- Причина: не поддерживать две параллельные модели поведения.

## 2026-07-22 — Работа вертикальными срезами

После каждого слоя физики собирается небольшой рабочий сценарий поведения. Полную физику заранее не строим.

## 2026-07-22 — Один активный подпроект

- Активен только `infantry-combat-prototype-v1`.
- Старые подпроекты остаются историческими материалами и не участвуют в текущем статусе.

## 2026-07-22 — Документация не блокирует разработку

- Повреждённый основной JSON или отсутствие активного подпроекта остаются ошибками.
- Старые ссылки, пути, статусы, SHA, версия PixiJS и устаревшая генерация дают предупреждения.
- Проверка документации не запускает проверки игры или производительности.
