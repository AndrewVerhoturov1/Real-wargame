# Критерии первого прототипа

## Общая сцена

- [ ] Один отряд атакует, другой обороняется.
- [ ] Бой идёт без ручного управления отдельными бойцами.
- [ ] Бойцы используют только доступные им сведения.
- [ ] Решения ИИ видны в диагностике.
- [ ] Поведение не мигает и не меняется каждый такт.
- [ ] Бой завершается победой одной стороны.

## Физическая основа

- [ ] Позиции наблюдения, защиты и огня различаются.
- [ ] Поза меняется во времени: стоя, пригнувшись, лёжа.
- [ ] Поза влияет на движение, видимость, укрытие и стрельбу.
- [ ] Действия сохраняются и восстанавливаются.

## Стрелковый бой

- [ ] Винтовка, ППШ и пулемёт ведут себя по-разному.
- [ ] Работают прицеливание, очередь, перезарядка и отмена огня.
- [ ] Учитываются поза, движение, дальность, укрытие и союзники.
- [ ] Попадания и близкие пролёты создают подавление.

## Поведение

- [ ] Обороняющийся занимает позицию, наблюдает и ведёт огонь.
- [ ] Атакующий выбирает путь, реагирует на контакт и продолжает приказ.
- [ ] Пулемёт создаёт подавление, стрелки используют его результат.
- [ ] Отряд не выглядит набором независимых одиночек.
