# Первый прототип пехотного боя

## Цель

Один пехотный отряд атакует другой. Оружие: винтовки, ППШ, пулемёт. Главная проверка — понятное поведение ИИ в атаке и обороне.

## Принцип

Код даёт физику, факты и действия. Graph v2 выбирает поведение. Работа идёт короткими вертикальными срезами.
